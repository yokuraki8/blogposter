# Blog Poster - AI駆動型ブログ記事自動生成WordPressプラグイン

## プロジェクト概要

Blog Posterは、企業サイトのオウンドメディア運営を支援するための、AI駆動型ブログ記事自動生成WordPressプラグインです。

### 主な特徴

- **日本語品質へのこだわり**: 既存プラグインの課題を解決し、自然で高品質な日本語記事を生成
- **RAG機能**: 既存コンテンツを活用した記事生成と内部リンク自動生成
- **SEO最適化**: Yoast SEO連携による完全なSEO最適化
- **自動スケジュール**: WordPress Cronによる自動記事生成
- **画像生成**: AIによるFeatured Image自動生成

## 技術スタック

- WordPress 6.0以上
- PHP 8.0以上
- OpenAI API / Anthropic Claude API
- MySQL 5.7以上 / MariaDB 10.3以上
